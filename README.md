# Installation

I use vim-plug for this config, therefore we must download vim plug
with this command:

```bash
sh -c 'curl -fLo $HOME/.config/nvim/autoload/plug.vim --create-dirs \
       https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim'
```
Consult [vim-plug](https://github.com/junegunn/vim-plug) repository for more info.

Then run nvim and put `:PlugInstall` exit nvim and go back in and that's it!
